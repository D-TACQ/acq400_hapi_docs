fs_delay_gen

Usage
usage: fs_delay_gen.py [-h] [--d1 D1] [--d2 D2] [--ds DS] [--tpi TPI] [--tns TNS] [--frame FRAME] [--line LINE] [--kludge_flyback KLUDGE_FLYBACK] [--rawscale RAWSCALE]

Named Arguments

--d1

first delay (ticks)

Default: 4

--d2

last_delay+1 (ticks)

Default: 21

--ds

delay step (ticks)

Default: 1

--tpi

ticks per interval

Default: 30

--tns

tick nsec

Default: 16.6666

--frame

frame line count

Default: 100

--line

line pixel count

Default: 100

--kludge_flyback

fixup for GPG logic

Default: 0

--rawscale

scale raw gpg output for slower output easier to see DO NOT USE!

Default: 1

Outline:

generate a series of variable delays for GPG and plot
